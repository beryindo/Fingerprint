* ID fingerprint dimulai dari 0.
* Tambahkan/enroll 6 jari (dari ID 0 sampai 5) untuk verifikasi
* Tambahkan/enroll 3 jari (dari ID 6 sampai 8) untuk enroll (menambahkan jari baru)
